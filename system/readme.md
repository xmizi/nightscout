# Systém
Nebudu zde uvádět konfiguraci a nastavení krok za krokem. Protože jde o server ve vlastní správě, měl by člověk něco málo znát. 
Takže základy typu instace balíčků, editace souborů, restart služeb, kde najít logy nebudu popisovat. Kromě nastavení firewallu (kvůli omezení přístupu k systému z důvěryhodných IP adres) není třeba žádné větší instalace. Doporučuji nastvait automatické upgrady systému - v Debianu jde o unattended-upgrades
